## Overview

This repository contains the code and data accompanying the paper **“Data-Driven Nonlinear Buckling Prediction and Inverse Design of Composite Gradient Lenticular Tubes”**.

A geometry-constrained, trajectory-centered machine-learning framework is developed for the prediction and inverse design of nonlinear buckling responses of composite gradient lenticular tubes (CGLTs). The workflow integrates high-fidelity finite-element simulations, surrogate modelling, sensitivity analysis, and multi-objective optimisation to enable fast and manufacturable design of deployable composite tubular structures.
